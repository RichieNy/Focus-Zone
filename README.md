## Project Overview  
Focus Zone is a web application designed to help users find the perfect place to study. Users can search for study spots based on specific preferences (such as bakeries, coffee shops, or quiet spaces), view results on an interactive map, and get directions to their chosen location.

---

## Goal  
The goal of this project is to create an intuitive and interactive application where users can:
- Search for study-friendly locations using filters  
- View results visually on a map  
- Navigate to their selected destination  

---

## How It Works  
1. The user enters a search query or selects filters (for example, type of place).  
2. Location data is retrieved using Geoapify.  
3. Results are displayed as markers on a Mapbox interactive map.  
4. Each location includes images pulled from Google Places.  
5. Users can view details and navigate to the selected study spot.  

---

## APIs Used  

### Geoapify  
Provides location search with filtering options and returns places based on user input.

### Mapbox  
Displays interactive maps, places markers for each study location, and provides navigation and routing features.

### Google Places  
Supplies images for each location to enhance the user experience.
